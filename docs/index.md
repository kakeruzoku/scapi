# Scapiとは
ScapiはScratch APIを非同期に実行できるPythonモジュールです。

# なぜScapi?
- **非同期処理を使用した効率的な実行**
- **ほぼすべて**のAPIをサポート
- **型ヒント**による開発のしやすさ
- オープンソース
