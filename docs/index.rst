バージョン3.x.xへの移行
-----------------------

バージョン ``3.0.0`` でScapiは完全に書き直されました。 ``2.x.x``以前を使用している場合、コードの大幅な変更が必要になります。詳しくは :doc:`このページ <migrating>` を参照してください。
