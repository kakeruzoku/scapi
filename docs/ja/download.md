# Scapiのダウンロード
## Pythonのダウンロード
[ここ](https://www.python.org/downloads/ )からダウンロードしてください。(推奨:最新版)

## Scapiのダウンロード
Scapiをインストールするには、ターミナルで次のコマンドを実行します:
```
python -m pip install scapi
```

## Scapiのアップデート
Scapiをアップデートするには、ターミナルで次のコマンドを実行します:
```
python -m pip install -U scapi
```
